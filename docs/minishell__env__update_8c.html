Adds an assigned environment variable to the new environment array.

Formats the variable into a KEY=VALUE string and stores it in new_env. If memory allocation fails, the function returns false.

Parameters
varPointer to the environment variable structure.
new_envThe array where the formatted entry will be stored.
idxPointer to the current index in new_env.
